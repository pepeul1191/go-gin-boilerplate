## Go-Gin

Instalación de dependencias:

    $ go get github.com/gin-gonic/gin
    $ go get -u github.com/jinzhu/gorm

Las variables globales deben empezar con mayúscula

Autoreload ante cambios:

    $ go get github.com/codegangsta/gin
    $ gin run main.go
